MirrorWealth Dashboard - Vite + React + Node.js

Quick Start:
```bash
npm install
npm run dev
```

The app runs at http://localhost:3000 with the dark-themed dashboard.

Project Structure:
- src/main.tsx - React entry point
- src/App.tsx - Main layout with React Router
- src/pages/ - 8 page components (Home, Dashboard, Allocation, Analytics, Backtesting, Insights, Rebalance, Settings)
- src/components/ - Reusable components (Header, Sidebar, KPICards, AllocationChart)
- src/index.css - Global styles with Tailwind CSS v4
- vite.config.ts - Vite configuration
- index.html - HTML entry point

Features:
- Dark theme (black background + orange accents)
- React Router v7 navigation
- Responsive design
- TypeScript support
- Tailwind CSS v4
